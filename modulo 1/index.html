Para utilizar Negrito e Italico

Para utilizar o Negrito basta utilizar a tag STRONG (possui semantica)

Para Utilizar a italico basta utilizar a tag EM (possui semantica)

Marcação igual marca texto

Para marcar um texto como se fosse um marca texto
igual esse texto marcado utiliza a tag <mark>.

Mas se voce quiser uma marcação de outra cor como essa por exemplo voce tem que fazer o comando de background-color

Agora se voce que uma marcação com todos da mesma cor mas sem prejudicar outro texto grifado, voce vai precisar fazer uma tag <Style>
